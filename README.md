# react-webpack-test

Repository with minimal ReactJS stack test
* reactjs 0.14
* redux
* react-router & redux-router
* redux-devtools
* react transform for hot-reloading and errors
* es6 components
* sass scss
* webpack
 * bundling
 * babel loader for es6 & es7 -> es5
 * sass loader - sass boostrap loader (no js)
 * webpack-dev-server
